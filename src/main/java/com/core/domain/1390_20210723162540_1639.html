三、施工准备及双方权利义务

   （一）甲方工作及权利义务

1．甲方开工前，应当到房屋所在区建设局监察大队或其委托的物业管理部门办理装修备案手续并由其支付相应费用。装修过程中，禁止违法拆改、变动承重结构和建筑主体；禁止违法超过设计标准或规范，增加房屋使用荷载；禁止卫生间、厨房间移位、扩大或增设；不得擅自改变房屋外立面、搭建建筑物、构筑物、改变房屋使用性质。接受并服从物业的统一管理和指导监督。

在确保安全和不违规的原则下，确需改变房屋结构和使用性质的，应当请具备相应资质的单位出具鉴定和加固方案，经区县（市）房产主管部门办理装修备案手续，领取《住宅装修许可单》，方可施工。

同时，甲方对施工图纸、工程预算协议书予以认可后，签订合同，并按合同规定向乙方支付工程款或其他相关款项。

3．做好施工中临时性使用公用部位的操作工作，以及产生影响邻里关系等行为的协调工作。非业主的住宅使用人，还需提供业主同意装饰装修的书面证明。

五、安全生产

施工图纸、施工说明及施工现场应符合防火、防事故的要求，主要包括电气、通讯线路、煤气管道、自来水和其它管道畅通、合格。乙方在施工中应采取必要的安全防护和消防措施，保障作业人员及相邻居民的安全，防止相邻居民住房的管道堵塞、渗漏水、停电、物品损坏等事故的发生。如遇上述情况发生，属甲方责任的，甲方负责修复或赔偿；属于乙方责任的，乙方负责修复或赔偿。在装修过程中形成的各种固体、可燃液体等废物，应当按规定的位置、方式和时间堆放和清运。若违反有关安全操作规程、消防条例，导致发生安全或火灾事故，乙方应承担由此引发的经济损失。乙方应安全生产，文明施工。由于乙方原因导致施工人员发生工伤及死亡事故均由乙方自行负责。

六、施工内容和工期的变更

1.乙方严格按施工图纸、工程预算协议书施工。

2.在施工过程中，甲方提出设计修改意见及增减工程项目时须提前与乙方联系，双方在签订《工程项目变更单》后（详见附件四），乙方才能进行该项目施工。因甲方原因变更导致的经济支出和乙方损失，由甲方承担，工期相应顺延。凡甲方私自与工人商定更改施工内容所引起的一切后果，甲方自负，给乙方造成损失的，甲方应予赔偿。

3.在施工过程中，甲方以口头、书面等方式向乙方提出设计异议或增减项目、变更材料的，乙方有权暂停施工，并要求甲方到乙方公司办理变更手续，及时确认工程量、价款及工期，未办理前不予恢复施工，由此造成乙方损失由甲方承担。因乙方设计错误导致甲方提出的设计异议，由乙方承担损失。

4．因以下原因造成竣工日期延误的，由甲方承担由此延误的工期和增加的费用：（1）甲方原因导致工程量变化或设计变更；（2）甲方未按合同约定支付工程价款的；（3）甲方未按合同约定提供施工条件；（4）由甲方原因引起工期顺延的其他情况。

5.因乙方责任不能按期完工的，工期不顺延，乙方按约定承担违约责任。

3．由于甲方原因造成的工程质量问题，其返工费用由甲方承担，工期顺延。

4．由于乙方原因造成质量事故，其返工费用由乙方承担，工期不变。

5．甲方对工程存在的问题应及时向乙方工程部反映投诉，以免给双方造成更大的返工或损失。

8．如甲、乙双方有质量争议，由法定质检机构进行检验，费用由责任方承担。

八、违约责任

1．本合同签订生效后，甲乙双方应履行合同所规定的各项条款（包括经双方确认的合同附件、预算及图纸等）。合同签订后施工前，任何一方若要终止合同，应以书面形式提出，经相对方同意后，办理终止合同手续，并由责任方按合同总价款10%支付违约金。施工过程中任何一方无理由提出终止合同，应以书面形式提出，经相对方同意后办理清算手续，并由责任方按合同总价款20%赔偿。

2．由于甲方原因导致延期开工或中途停工，甲方应补偿乙方因停工、窝工所造成的损失，每延长一天，甲方支付乙方      100    元，工期顺延。

3.由于乙方原因逾期竣工的，每逾期一天，乙方支付甲方       200      元违约金。

4.甲方未按合同的约定付款的，每逾期一天，按     100    元支付违约金，工期顺延。

5．甲方未办理任何手续，要求乙方改变房屋结构和使用性质，乙方不得施工；若甲方坚持，由此发生的损失或事故（包括罚款）由甲方负责承担。

6．未经甲方同意，乙方擅自改变房屋结构和使用性质，由此发生的损失或事故（包括罚款），由乙方负责承担。

7. 工程未办理验收、结算手续，甲方提前使用或擅自动用该工程成品或自行入住，由此造成无法验收及损失的，由甲方承担。

8. 甲方支付设计服务费后，由乙方向甲方交付设计图纸。如因甲方原因导致合同提前终止的， 则设计服务费不予退还；若因乙方设计原因导致合同终止的，则乙方须退还已支付的设计服务费， 若因乙方的施工原因导致合同提前终止的，则乙方在扣除必要的设计成本后，退还甲方剩余已支 付的设计服务费。

圣都零增项客户知情书

一、为了防止和杜绝“诱导增项”、“低吸高结”现象，本着“为客户建设一个温暖的家”的服务宗旨，公司经研究决定，实行零增项规定。  

二、圣都零增项基本原则：从客户实际装修需求出发，充分保障客户装修权益，让客户在装修过程中享有消费知情权，圣都公司在设计、预算、材料、施工等全方面实施零增项，绝不恶意增项及不合理增项。

三、零增项控制节点主要为四个：

（一）设计零增项；

（二）预算零增项；

（三）材料零增项；

（四）施工零增项。

四、设计零增项主要指标：

1.施工图与现场面积相符，不因面积误差增项；

2.施工图、装修布置图标注清晰，不因标注失误增项；

3.施工图、装修布置图施工项目齐全，不存在故意拉项、减项；

4.施工图、装修布置图应设计合理，不因图纸问题增项。

五、预算零增加主要指标：

1.预算项目和备注标准清晰，不存在误解；

2.不存在少算、漏算、错算。

六、材料零增项主要指标：

1.对于精装内标配材料，不能增项；

2.对于展厅出样的材料标签错写/漏写，不能增项；

七、施工零增项主要指标：

1.设计施工图不变，不按图纸、预算内项目施工而产生的项目不增项；

2.精装内厨房、卫生间、阳台包管道不增项；

3.精装内墙顶面乳胶漆、墙面网格布，无特殊要求的，不增项；

4. 材料进场不与客户确认，而产生的返工、整改费用不增项；

八、发生以下情形而增加的工程量，不属于恶意增项，工程量及施工项目应按实结算：

（一）客户自行提出设计变更；

（二）客户自行调整或改变装修预算；

（三）客户对装修材料、空间格局，使用功能、施工工艺等作出调整要求；

（四）客户提出的其他有关设计、预算、材料、施工方面等的变更，超出装修设计及预算范围；

九、若客户发现圣都公司人员不遵守零增项规定的，可以向圣都公司客户服务中心投诉，也可向圣都公司董事长颜伟阳个人公众号反映。
