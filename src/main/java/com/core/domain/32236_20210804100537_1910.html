四、计算细则说明

1、精装内提供的整体厨柜下柜和台面，超出部分按增项计算，不足部分不给于减项；吊柜按底柜长度的0.5倍计算，超出部分按增项计算，不足部分不给于减项。

2、精装内提供的标配木门，超出标准尺寸，高、宽、厚每项超出部分按20元/cm收费；双开门的费用以标准门的2倍另计；推拉门加600元/樘，滑道费用另计；子母门的费用以标准门的2倍另计。

3、精装提供每个客户3.36平方厨房金属移门+5.8米门套，甲方（客户）可选择免费换成标配木开门1樘。

4、厨房、卫生间墙面砖铺贴高度在2.4米以内,超高另计.扩大面积部分费用需另行计算。

5、精装内的厨房和卫生间的铺砖、集成吊顶等面积都是根据原始结构图上的原始面积大小计算的，如果后期根据客户设计需求，需要扩大原始厨房或卫生间的面积，则扩大面积部分的材料及人工费用需要另行计算。

6、精装内不含阳台墙砖，如原始阳台要包进卧室或客餐厅或厨卫，则地板或地砖或墙砖的补差费用需要另行计算。

7、精装内的阳台各含一个地漏，卫生间含二个地漏，精装内的洁具包含三角阀和软管。

8、参加精装，如遇墙面超高或顶面为三角坡顶的超出部分，按实际面积根据乙方（公司）标准报价收取；如遇墙面高度超过3米以上，必须租借脚手架施工的情况，此费用由甲方（客户）自理。

9、若房间中有新建楼层，所有费用另行核算。
